## 经验与决策

- Android 9+ 应用默认拦截 CLEARTEXT 明文 HTTP 请求，连接 WebDAV 需处理此限制
- WebDAV 连接时 405/401 错误通常由路径错误或认证问题引起
- 连接局域网服务时应使用电脑的局域网 IP（如 192.168.1.105）而非 127.0.0.1 回环地址
- 5244 端口服务为 AList，其 WebDAV 路径为 /dav
- AList 的 WebDAV 中文目录显示异常：路径重复（/dav/dav/）且 127.0.0.1 未替换为正确 IP
